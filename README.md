## Como iniciar

#### Com yarn:

```bash
yarn start
```

#### Com npm:

```bash
npm run start
```

## Para instalar todas as dependências é necessário usar

#### Com yarn:
```
yarn install
```
#### Com npm:
```
npm install
```
